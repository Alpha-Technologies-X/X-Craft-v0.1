# XCraft

XCraft is an open-source browser-based voxel sandbox game inspired by early block-building games.

## Features
- JavaScript OR WebAssembly engine selection
- Browser-based multiplayer using PeerJS
- GitHub Pages compatible
- No proprietary assets or code
- Open-source
